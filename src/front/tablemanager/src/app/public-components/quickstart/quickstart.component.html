Quickstart

Here is a quick overview of the application and its features.

Create a PDF

Upload a logo from your computer (the logo must be .png extention) and fill in your PDF info.


Entries

1. Set VAT percentage, 2. Add an entry, 3. Edit entry information, 4. Add entry to pdf generation list or generate PDF, 5. Delete entry

Edit entry

By clicking edit entry, you can modify the entry.

Add entry to PDF generation list or generate PDF

1. Add an item to pdf generation list. If you want multiple items to list, reopen the window on desired item and add it to list.

2. Fill in the customer info and generate the PDF. PDF will be downloaded to your computer.
